Final Project

Our final project, was creating a game directory that has many different types mini games in it. Our directory holds six different games: a Memory Tile matching game, Frogger, TIC-TAC-TOE, Pokemon battles, Dodgeball, and Fruit Ninja. From the directoy you can select the game you want to play where it will then bring you to an instructions page telling you how to play the game. Then you are prompted to start the game. Every game can be restarted and tried again after either hitting a games restart button or refreshing the page. Now go have fun and watch the tutorial video if you need help!
